# 🤝 Contributing to AI RAG Assistant

Спасибо, что хочешь помочь проекту! 🎉

## 🧭 Как начать

1. **Форкни** репозиторий 🍴
2. Создай ветку для фичи: `git checkout -b feature/awesome-thing`
3. Делай изменения — и пусть код будет чистым ✨
4. **Запуши** и открой Pull Request 🚀

## 🧪 Code Style

- 🐍 Python — соблюдай PEP 8
- 🐚 Shell — проверь `shellcheck`
- 📝 Комментарии — по делу, без мусора
- 😊 Эмодзи в коммитах приветствуются (но без фанатизма)

## 💬 Commit Messages

Формат: `тип: краткое описание`

```
feat: add RAG pipeline for markdown files
fix: resolve Docker build timeout
docs: update README with usage examples
```

Типы: `feat`, `fix`, `docs`, `refactor`, `test`, `chore`

## 🐛 Как сообщить о баге

Открой [Issue](https://github.com/trollimo/ai-rag-assistent/issues/new) и укажи:

- ✅ Шаги воспроизведения
- ✅ Ожидаемое vs реальное поведение
- ✅ Окружение (ОС, Python, Docker версии)
- ✅ Логи/скриншоты (по желанию)

## 🧱 Структура проекта

```
rag-generation/        — Python-скрипты для генерации RAG
assistent-container/   — Docker с Web и MCP
```

## 📬 Вопросы

Пиши в Issues или обсуждения. Будем рады 💬
